# A tool for reverse engineering 3rd party, closed, binary Android apps.

## Input

```
.apk-files and .jar-files

```

## Output

```
Folder with decompiled application files
```

## Usage

```
docker run --rm -v `pwd`:/data cincan/apktool:dev d apk_file.apk

```
