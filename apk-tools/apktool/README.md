# A tool for reverse engineering 3rd party, closed, binary Android apps.

## Input

```
.apk-files and .jar-files

```

## Output

```
Apk tool report
```

## Usage

```
docker run --rm -v `pwd`:/data apktool d apk_file.apk

```
